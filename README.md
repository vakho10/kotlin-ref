# kotlin-ref

A personal blog built with [Jekyll](https://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/).

**Live site:** https://vakho10.github.io/kotlin-ref/

## Local Development

```bash
cd docs
bundle install
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000/kotlin-ref/`.

## Project Structure

```
docs/           # Jekyll source files (deployed via GitHub Pages)
  _posts/       # Blog posts (Markdown with YAML frontmatter)
  _config.yml   # Jekyll site configuration
  Gemfile       # Ruby dependencies
CLAUDE.md       # Claude Code instructions
README.md       # This file
```
